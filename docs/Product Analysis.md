# 产品功能分析

## 概述

本文档介绍 Mall 系统的核心功能模块，帮助团队全面了解系统的能力和业务覆盖范围。

## 功能模块

### 1. 用户管理

- 用户登录 / 登出
- 用户信息管理（增删改查）
- 用户角色与权限管理

### 2. 商品管理

- 商品列表展示
- 商品搜索与筛选
- 商品详情查看
- 商品新增 / 编辑 / 删除

### 3. 订单管理

- 订单创建
- 订单列表与详情
- 订单状态流转（待支付 → 已支付 → 已发货 → 已完成）
- 订单取消与退款

### 4. 购物车

- 添加商品到购物车
- 修改购物车商品数量
- 移除购物车商品
- 购物车结算

### 5. 支付管理

- 支付方式选择
- 支付状态回调
- 支付记录查询

### 6. 后台管理

- 管理员登录
- 数据仪表盘
- 系统配置管理

## 功能矩阵

| 功能模块 | 前台用户 | 后台管理员 | 备注 |
|---------|---------|-----------|------|
| 用户登录 | ✅ | ✅ | 不同角色不同入口 |
| 商品浏览 | ✅ | ✅ | 管理员可编辑 |
| 订单操作 | ✅ | ✅ | 权限区分 |
| 购物车 | ✅ | ❌ | 仅前台用户 |
| 数据统计 | ❌ | ✅ | 仅后台管理 |
