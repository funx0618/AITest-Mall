# 页面业务流程

## 概述

本文档描述 Mall 系统中用户完成各项功能的操作流程，涵盖主要业务场景的页面跳转和交互步骤。

## 1. 登录流程

```
┌──────────┐    输入账号密码     ┌──────────┐    验证成功     ┌──────────┐
│ 访问首页  │ ────────────────→ │ 登录页面  │ ────────────→ │ 系统首页  │
└──────────┘                   └──────────┘               └──────────┘
                                      │
                                      │ 验证失败
                                      ↓
                               ┌──────────┐
                               │ 错误提示  │
                               └──────────┘
```

**操作步骤：**
1. 访问系统首页，点击「登录」
2. 输入用户名和密码
3. 点击「登录」按钮
4. 验证成功 → 跳转到系统首页
5. 验证失败 → 页面显示错误提示

## 2. 用户管理流程（后台）

### 2.1 用户列表查看

```
后台首页 → 用户管理 → 用户列表
```

**操作步骤：**
1. 管理员登录后台
2. 进入「用户管理」模块
3. 查看用户列表（支持分页、搜索、筛选）

### 2.2 新增用户

```
用户列表 → 点击「新增」 → 填写用户信息 → 提交 → 返回列表（显示新用户）
```

**操作步骤：**
1. 在用户列表页点击「新增用户」按钮
2. 填写用户名、密码、角色等信息
3. 点击「提交」
4. 成功 → 返回列表，显示新增用户
5. 失败 → 页面提示错误信息

### 2.3 编辑用户

```
用户列表 → 点击「编辑」 → 修改用户信息 → 提交 → 返回列表（信息已更新）
```

**操作步骤：**
1. 在用户列表中找到目标用户
2. 点击「编辑」按钮
3. 修改用户信息
4. 点击「提交」
5. 成功 → 返回列表，信息已更新

### 2.4 删除用户

```
用户列表 → 点击「删除」 → 确认弹窗 → 确认 → 返回列表（用户已移除）
```

**操作步骤：**
1. 在用户列表中找到目标用户
2. 点击「删除」按钮
3. 弹出确认对话框
4. 点击「确认」
5. 成功 → 返回列表，用户已移除

### 2.5 用户搜索

```
用户列表 → 输入搜索条件 → 点击「搜索」 → 显示匹配结果
```

**操作步骤：**
1. 在搜索框输入关键词（用户名等）
2. 点击「搜索」按钮
3. 列表显示匹配结果
4. 点击「重置」清空搜索条件

## 3. 购物流程

```
商品列表 → 商品详情 → 加入购物车 → 购物车 → 结算 → 确认订单 → 支付 → 完成
```

**操作步骤：**
1. 浏览商品列表或搜索商品
2. 点击商品进入详情页
3. 选择规格、数量，点击「加入购物车」
4. 进入购物车，确认商品信息
5. 点击「结算」
6. 确认收货地址和订单信息
7. 选择支付方式并完成支付
8. 订单创建成功

## 4. 订单流程

### 4.1 订单状态流转

```
待支付 → 已支付 → 已发货 → 已完成
   │
   └→ 已取消（超时或用户取消）
```

### 4.2 查看订单

```
个人中心 → 我的订单 → 订单列表 → 订单详情
```
